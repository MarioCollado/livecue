# LiveCue

LiveCue es una herramienta enfocada en el manejo de cues desde Ableton en tiempo real. Está desarrollada en Python y utiliza Flet y Python-OSC para la interfaz y comunicación en tiempo real.

---

## Características

- Interfaz gráfica moderna con **Flet**.
- Comunicación en tiempo real usando **Python-OSC**.
- Fácil de ejecutar en Windows mediante PyInstaller.
- Configuración mínima para comenzar a usarlo.

---

## Requisitos

- Python 3.13+
- [Flet](https://pypi.org/project/flet/)
- [Python-OSC](https://pypi.org/project/python-osc/)
- Pillow (para manejo de imágenes)

Puedes instalar las dependencias con:

```bash
pip install -r requirements.txt
